    Skll

    This was a personal project to solve a problem I was having when making job applications. Recruiters are increasingly using applicant tracking programmes to judge suitability and filter out many cv's before a human ever gets to see them. CV's can fail on incorrect formatting or missing keywords, so this web application runs your cv text against the job application and makes suggestions on what you can include to make it perform better.

    The backend uses Python NLP module 'Spacy' to parse the text and pickout words from a large skills library stored in the SQLite3 database. There is a login option where you can store your cv information and even search for local jobs based on the skills in your profile.

    NLP | Python | SQLite3 | Flask | Javascript | HTML | Heroku

    Onboard

    Designed and built in collaboration with a large financial services business, Onboard is a powerful web application for the better training and tracking of new employees. It was built to modernise their outdated, spreadsheet driven enrollment processes and allow senior managers better visibility across the whole workforce. All this data is brought together to make predictions on the success of a particular individual, their graduation date and where they sit in relation to their peers.

    Built in Python on the Flask Framework, Onboard uses a SQLite3 database to manage logins performance tracking across multiple tiers and all the pathway data associated.

    Python | SQLite3 | Flask | Javascript | HTML | Heroku

About Me
